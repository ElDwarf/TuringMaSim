# TuringMaSim

Is a simple turing machine simulator.

## How Install ##
 1. Create virualenv 
    ```shell
    $ virtualenv venv -p python3
    ```   
 2. Install requirement.txt
    ```shell
    $ pip install -r requirement.txt
    ```
 3. Go to src 
    ```shell
    $ cd src
    ```
 4. run main.py
    ```shell
    $ python main.py
    ```
